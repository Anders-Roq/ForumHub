## 📚 ForumHub API

O **ForumHub** é uma API REST desenvolvida com **Spring Boot** que simula o funcionamento de um fórum de discussões. A aplicação permite o gerenciamento de tópicos, incluindo criação, listagem, atualização e remoção.

A API também implementa **autenticação com JWT**, garantindo que apenas usuários autenticados possam acessar os endpoints protegidos.

O projeto foi desenvolvido como parte de um desafio de backend, com foco em boas práticas de desenvolvimento, arquitetura em camadas (Controller, Service e Repository) e integração com banco de dados relacional.

### 🚀 Tecnologias utilizadas

- Java
- Spring Boot
- Spring Data JPA
- Spring Security
- JWT (JSON Web Token)
- H2 Database
- Maven
